# نظام معلومات الطلاب - جامعة القاهرة الجديدة التكنولوجية (NCTU SIS)

نظام معلومات طلاب شامل مبني باستخدام Qt/C++ و MySQL لإدارة البيانات الأكاديمية والإدارية في الجامعة.

## المميزات

- **تسجيل دخول متعدد الأدوار**: طالب، إداري، دكتور
- **بوابة الطالب**: عرض الملف الشخصي، المقررات، الدرجات، الحضور، الدفع، والأخبار
- **لوحة تحكم الإدارة**: إدارة الطلاب، المقررات، التسجيلات، الدرجات، والدفع
- **لوحة تحكم الدكتور**: عرض المقررات المخصصة، إدخال الدرجات، وتسجيل الحضور
- **نظام الدرجات**: حساب تلقائي للمجموع والدرجة الحرفية
- **نظام الحضور**: تتبع الحضور مع تحذيرات تلقائية عند تجاوز الحد المسموح

## المتطلبات

- **Qt 5.15+ أو Qt 6.x**
- **CMake 3.16+**
- **MySQL Server 5.7+ أو 8.0+**
- **مترجم C++17** (GCC, Clang, أو MSVC)

## التثبيت والإعداد

### 1. إعداد قاعدة البيانات

```bash
# إنشاء قاعدة البيانات والجداول
mysql -u root -p < db/mysql_create.sql

# إدخال بيانات تجريبية
mysql -u root -p < db/seed_data.sql
```

### 2. بناء المشروع

```bash
# إنشاء مجلد البناء
mkdir build
cd build

# تكوين المشروع
cmake ..

# البناء
cmake --build .

# أو على Windows مع Visual Studio
cmake --build . --config Release
```

### 3. تشغيل التطبيق

```bash
# Linux/macOS
./NCTU_SIS

# Windows
NCTU_SIS.exe
```
